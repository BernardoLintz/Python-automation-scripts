# Python Automation Scripts

## Português

Este repositório contém scripts Python para automação de tarefas e análise de segurança. Ele está organizado nas seguintes categorias:

**/task_automation**: Scripts para agendamento e automação de tarefas.
 
**/network_logs_collection**: Scripts para coleta de logs de rede.

**/vulnerability_detection**: Scripts para detecção de vulnerabilidades.

**/port_scanning**: Scripts para varredura de portas de rede.

## English

This repository contains Python scripts for task automation and security analysis. It is organized into the following categories:

**/task_automation**: Scripts for scheduling and automating tasks.

**/network_logs_collection**: Scripts for network log collection.

**/vulnerability_detection**: Scripts for vulnerability detection.

**/port_scanning**: Scripts for network port scanning.

## Español

Este repositorio contiene scripts en Python para automatización de tareas y análisis de seguridad. Está organizado en las siguientes categorías:

**/task_automation**: Scripts para agendar y automatizar tareas.

**/network_logs_collection**: Scripts para la recopilación de registros de red.

**/vulnerability_detection**: Scripts para la detección de vulnerabilidades.

**/port_scanning**: Scripts para escaneo de puertos de red.
